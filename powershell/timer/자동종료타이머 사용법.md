## 사용법
1. powershell.bat 파일과 shutdownCounter.ps1 파일을 같은 폴더에 둔다.  
    > 경로명에 한글이 있으면 정상적으로 작동하지 않는다. -solved-
2. powershell.bat 파일 더블클릭으로 실행
3. 원하는 종료시간을 입력한다.  
    > 형식 -> 20220520 2359  
4. 취소하려면 Ctrl + C 누른다.
